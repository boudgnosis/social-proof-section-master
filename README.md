### Lo que aprendi

Aprendi a crear un layout básico con **CSS GRID** y a poder posicionar los lementos usando esto mismo.

```css
body {
		display: grid;
		grid-template-columns: repeat(3, 1fr);
		grid-template-rows: repeat(2, 1fr);
}
```

```css
.container {
		grid-column: 1 / 3;
}
```
